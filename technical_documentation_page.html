Technical 1

Here is how you link a stylesheet.

link rel="stylesheet href="styles.css"
  • Make sure there are no typos.
  • Make sure you enclose it in < >.
  • Make sure the values are in " ".
  • Make sure your href value matches the name of your stylesheet.
  • Make sure your browser extensions are disabled.

    • That is how you link a stylesheet.

    Technical 2

    Here is how you set a charset.

    meta charset="utf-8"

    That sets the charset to UTF-8.

    Technical 3

    Here is how you set the viewport.

    meta name="viewport" content="width=device-width, initial-scale=1.0"

    That is how you set the viewport.
